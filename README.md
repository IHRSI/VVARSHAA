# VVARSHAA 💔

> 8 idiots. 1 group chat. 55,000+ messages. Infinite memories.

A tribute website for the VVARSHAA friend group — MIT Manipal, Batch of '27.

## Features
- 📊 **Chat Statistics** — 51,528 messages analyzed across 930 active days
- 👥 **Member Profiles** — Personalized descriptions for all 8 members
- 🏆 **Leaderboard** — Who talks the most (spoiler: Vedant & Aryan)
- 🕐 **Activity Heatmap** — When does the group peak (10PM-midnight obviously)
- 💬 **Legendary Chats** — The conversations that defined us
- 🗺️ **Timeline** — From "Eh idk" to VVARSHAA💔
- 🎵 **Original Songs** — The VVARSHAA Anthem + Rap Cypher
- 💎 **Quotes** — Words of wisdom from each member
- 🤡 **Meme Wall** — Awards nobody asked for

## Deploy
Push to GitHub → Connect to Vercel → Deploy from `site/` directory.

Set **Root Directory** to `site` in Vercel settings.
